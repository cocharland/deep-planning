About A* Algorithm

A* is an informed search algorithm that uses a heuristic function to guide its search. Unlike Dijkstra's, which explores equally in all directions, A* prioritizes paths that appear to be leading toward the goal.

Cost Function: f(n) = g(n) + h(n)

Where g(n) is the actual distance from start, and h(n) is the estimated distance to goal (heuristic).

How to Use:

  1. Select a heuristic function (Manhattan works best for grid movement)
  2. Click "Set Start" and click on the grid to place the start point (green)
  3. Click "Set Goal" and click on the grid to place the goal point (red)
  4. Click cells to add/remove walls (dark gray) - optional
  5. Click "Run A*" to visualize the algorithm finding the shortest path
  6. Compare the number of explored nodes with Dijkstra's algorithm!
